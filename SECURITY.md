# Security Policy

## Supported Versions

Only the current major version is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it privately to our security team.
Do **not** create a public GitHub issue for security vulnerabilities.

Please send your report to: **security@aegis.local**

We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress. If you do not hear back within 48 hours, please reach out again.
